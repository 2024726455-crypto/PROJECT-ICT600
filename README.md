# Badminton Court Booking System

This is a runnable prototype implementation based on the low-fidelity pages in the provided PDF.

## Pages

- `index.html` - login
- `register.html` - user registration
- `dashboard.html` - user dashboard
- `membership.html` - membership application
- `booking.html` - court booking
- `payment.html` - payment
- `availability.html` - check availability
- `history.html` - booking history

## How to Run

Open `index.html` in a browser, or serve this folder with any static web server.

The app stores sample runtime data in the browser's `localStorage`.
